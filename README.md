## 📖 Descripción

Landing page profesional para una clínica veterinaria ficticia. Diseñada para demostrar habilidades en **maquetación web responsive**, **interacciones con JavaScript** y **atención al detalle visual**.

### ✨ Características destacadas

- 🌓 **Modo oscuro/claro** con persistencia (localStorage)
- 📱 **Diseño mobile-first** adaptable a todos los dispositivos
- 🎯 **Botón flotante de CTA** (Call to Action)
- 🧩 **Grid y Flexbox** para layouts modernos
- 🎨 **Microinteracciones** (hover, transiciones suaves)

## 🛠️ Tecnologías utilizadas

| Tecnología | Uso |
|------------|-----|
| HTML5 | Estructura semántica y accesibilidad |
| CSS3 | Diseño responsive, variables CSS, Grid/Flexbox |
| JavaScript | Toggle modo oscuro, interacciones básicas |
| Font Awesome 6 | Iconografía profesional |

## 🎯 Lo que aprendí con este proyecto

- Implementar un **toggle de tema oscuro/claro** que recuerda la preferencia del usuario
- Organizar CSS con **variables** para facilitar el mantenimiento
- Crear un layout **responsive sin frameworks** (puro CSS)
- Diseñar pensando en la **experiencia de usuario** (botón de cita visible siempre)
